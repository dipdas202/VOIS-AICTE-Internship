🔍 Network Port Scanner GUI
A lightweight and fast TCP port scanner with a graphical user interface built using Python and Tkinter.

📌 Overview
This project is a simple yet powerful network port scanner that allows users to detect open ports on a target system. It is designed for beginners and students to understand networking concepts and cybersecurity basics.

🚀 Features
Simple GUI (Target, Start Port, End Port)
Multi-threaded scanning (fast performance)
Real-time progress bar and elapsed time
Detects common services (HTTP, SSH, FTP, etc.)
Stop scan anytime
Save results to a .txt file
Cross-platform (Windows, Linux, macOS)
🛠️ Technologies Used
Python
Tkinter (GUI)
Socket Programming
Multithreading
⚙️ Requirements
Python 3.7 or higher
Tkinter (comes with Python)
